## ADDED Requirements

### Requirement: Modern Navigation
The site layout SHALL include a sticky, translucent navigation bar with links to key sections.

#### Scenario: Sticky Navigation
- **WHEN** the user scrolls past the hero section
- **THEN** the navigation bar remains visible at the top of the viewport with a "glassmorphism" effect.

### Requirement: Footer
The site layout SHALL include a minimalist footer with copyright info and social links.

#### Scenario: Footer Visibility
- **WHEN** the user reaches the bottom of any page
- **THEN** they see a consistently styled footer.
