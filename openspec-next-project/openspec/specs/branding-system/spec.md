## Requirements
### Requirement: Brand Logo
The system SHALL feature a custom SVG logo for "BabyNails" in the navigation and hero sections.
(Previously: Lumina Nails & Art)

#### Scenario: Responsive Branding
- **WHEN** the user views the website on any device
- **THEN** they see a crisp, scalable SVG logo that maintains brand integrity.

### Requirement: Color Identity
The theme SHALL implement a premium palette of Cream (#FDFBF7), Rose Gold (#B76E79), and Soft Charcoal.

#### Scenario: Theme Application
- **WHEN** components are rendered
- **THEN** they utilize the custom Tailwind colors to ensure a cohesive luxury feel.
