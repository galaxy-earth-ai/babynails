## Requirements
### Requirement: Artistic Theme Palette
The theme SHALL use a sophisticated color palette (e.g., deep charcoal, vibrant accents like electric blue or gold) defined in Tailwind's configuration.

#### Scenario: Consistent Branding
- **WHEN** components use Tailwind utility classes
- **THEN** they reflect the custom theme's colors and spacing.

### Requirement: Typography
The theme SHALL use a combination of a clean sans-serif and a serif font for a "modern yet artistic" typographic hierarchy.

#### Scenario: Typography Hierarchy
- **WHEN** headings and body text are rendered
- **THEN** they use the specified variable fonts to maintain a professional and artistic look.
