## ADDED Requirements

### Requirement: Split-Screen Comparison
The system SHALL provide a "Compare Mode" that displays two different product overlays on the user's hand image.

#### Scenario: Activate comparison
- **WHEN** user clicks "Compare" button in the VTON Studio
- **THEN** system displays a vertical slider dividing the image into two sections, each with a different nail style.

### Requirement: Comparison Slider
The user SHALL be able to drag a vertical handle to adjust the visibility ratio of the two products.

#### Scenario: Drag handle
- **WHEN** user drags the center handle to the left
- **THEN** more of the right-side product is revealed.
