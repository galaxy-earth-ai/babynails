## ADDED Requirements

### Requirement: Shop Navigation Link
The main navigation SHALL include a link to the Shop page.

#### Scenario: Click Shop link
- **WHEN** user clicks "Shop" in the navigation bar
- **THEN** system navigates to the /shop page.

### Requirement: Cart Navigation Icon
The main navigation SHALL include a cart icon showing the number of items in the cart.

#### Scenario: Cart item count
- **WHEN** user adds 2 items to the cart
- **THEN** system displays the number "2" on the cart icon in the navigation bar.
